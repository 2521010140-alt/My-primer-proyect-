# My-primer-proyect-
Mi primer proyecto en GitHub. Estudiante de Ingeniería de Sistemas aprendiendo programación.

## SEO y despliegue

Para que Google encuentre tu sitio:

- Añade un dominio real y súbelo a un hosting.
- Usa `index.html`, `robots.txt` y `sitemap.xml`.
- Envía la URL a Google Search Console.
- Cambia `https://www.tusitio.com/` por tu dominio real en `index.html`, `sitemap.xml` y `robots.txt`.

## Instrucciones básicas

1. Registra un dominio como `tusitio.com`.
2. Sube los archivos a un host (GitHub Pages, Netlify, Vercel, etc.).
3. Verifica el sitio en Google Search Console.
4. Solicita indexación y espera unos días.
